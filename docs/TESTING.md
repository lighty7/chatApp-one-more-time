# Testing Overview

Quick testing guide. See [TESTING-detailed.md](TESTING-detailed.md) for comprehensive documentation.

## Quick Commands

```bash
# Run tests
npm test

# Watch mode
npm run test:watch

# Linting
npm run lint
npm run lint:fix

# Security audit
npm run security:check
```

## CI/CD Testing

- **GitHub Actions**: `.github/workflows/ci.yml`
- **GitLab CI**: `.gitlab-ci.yml`
