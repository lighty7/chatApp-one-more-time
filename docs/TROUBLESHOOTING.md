# Troubleshooting Overview

Quick troubleshooting guide. See [TROUBLESHOOTING-detailed.md](TROUBLESHOOTING-detailed.md) for comprehensive solutions.

## Common Issues

| Issue | Solution |
|-------|----------|
| MongoDB connection error | Check MONGODB_URI in .env |
| Redis connection error | Verify REDIS_HOST and REDIS_PORT |
| WebSocket issues | Check CORS settings |
| File upload fails | Check storage permissions |
